시작하기

Q1: KiwoomRestApi.Net 사용을 위한 필수 조건은 무엇인가요?

다음 사항들이 필요합니다:

  • .NET 6.0 이상
  • 키움증권 계정 및 API 접근 권한
  • 키움 OpenAPI 포털에서 발급받은 앱 키와 시크릿 키
  • Visual Studio 2022 또는 VS Code

Q2: 라이브러리를 어떻게 설치하나요?

NuGet을 통해 설치하세요:

dotnet add package KiwoomRestApi.Net

또는 패키지 관리자 콘솔에서:

Install-Package KiwoomRestApi.Net

Q3: 모의투자 모드와 실거래 모드의 차이점은 무엇인가요?

  • 모의투자 모드: 시뮬레이션된 데이터 사용, 실제 거래 없음, 테스트용 무료
  • 실거래 모드: 실시간 시장 데이터, 실제 거래 발생, 실제 계정 필요
팁: 개발 및 테스트 시에는 반드시 모의투자 모드부터 시작하세요.

API 사용법

Q4: 주식 정보를 어떻게 조회하나요?

StockInfo 클라이언트를 사용하세요:

var stockInfo = await client.StockInfo.GetStockInfoAsync(
    stockCode: "005930",
    date: DateTime.Today,
    marginLoanType: KiwoomStockInfoMarginLoanType.Loan
);

Q5: 대용량 데이터셋의 페이징은 어떻게 처리하나요?

ContYn과 NextKey 속성을 확인하세요:

do {
    var result = await client.SomeApiCallAsync();
    // Process result.Data
    client.NextKey = result.NextKey;
    client.ContYn = result.ContYn ? "Y" : "";
} while (result.ContYn && !string.IsNullOrEmpty(result.NextKey));

Q6: 동시에 여러 API 호출을 할 수 있나요?

네, 라이브러리는 완전히 비동기입니다. 단, 키움의 제한 속도를 고려하세요:

var tasks = new[]
{
    client.StockInfo.GetStockInfoAsync("005930", DateTime.Today),
    client.StockInfo.GetStockInfoAsync("000660", DateTime.Today),
    client.StockInfo.GetStockInfoAsync("035420", DateTime.Today)
};

var results = await Task.WhenAll(tasks);

인증

Q7: API 키는 어떻게 받나요?

  1. 키움 OpenAPI 포털에 로그인
  2. 애플리케이션 등록
  3. 앱 키와 시크릿 키 생성
  4. 모의투자 또는 실거래 환경 선택

Q8: 토큰을 직접 관리해야 하나요?

아니요, 라이브러리가 토큰 관리를 자동으로 처리합니다:

var client = await KiwoomRestApiClient.CreateAsync(appKey, secretKey, isMock);
// 토큰이 자동으로 획득되어 저장됩니다

Q9: 토큰이 만료되면 어떻게 하나요?

라이브러리가 토큰 갱신을 자동으로 처리합니다. 토큰 문제가 발생하면:

// 클라이언트 재초기화
await client.InitializeAsync();

거래

Q13: 매수 주문을 어떻게 체결하나요?

Order 클라이언트를 사용하세요:

var order = await client.Order.PlaceOrderAsync(
    orderType: KiwoomOrderType.Buy,
    exchangeType: KiwoomOrderDomesticStockExchangeType.Krx,
    stockCode: "005930",
    quantity: 10,
    transactionType: KiwoomOrderTransactionType.Market
);
중요: 거래 작업은 실제 API 키가 필요하며 신중하게 사용해야 합니다.

Q16: 모의 거래가 가능한가요?

네, 모의투자 모드에서 모의 거래가 가능합니다:

var client = await KiwoomRestApiClient.CreateAsync(appKey, secretKey, isMock: true);
// 모든 주문이 시뮬레이션됩니다

오류 처리

Q17: 일반적인 오류 코드는 무엇인가요?

  • OVERFLOW: 요청 과다
  • NOT_EXIST: 종목코드가 존재하지 않음
  • INVALID_PARAMETER: 유효하지 않은 API 파라미터
  • AUTH_EXPIRED: 인증 만료

Q19: 제한 속도에 도달하면 어떻게 하나요?

지수 백오프를 구현하세요:

int retryCount = 0;
int maxRetries = 3;

while (retryCount < maxRetries) {
    var result = await client.SomeApiCallAsync();
    if (result.IsSuccess) break;

    if (result.ErrorMessage?.Contains("OVERFLOW") == true) {
        await Task.Delay(1000 * (int)Math.Pow(2, retryCount));
        retryCount++;
    }
}
