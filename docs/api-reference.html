  • OnRealtimeStockBestQuoteReceived (Type: 0C) - 실시간 최우선 호가
  • OnRealtimeOrderBookReceived (Type: 0D) - 실시간 호가참
  • OnRealtimeAfterMarketQuoteReceived (Type: 0E) - 실시간 시간외 호가

  • OnRealtimeDailyBrokerReceived (Type: 0F) - 실시간 일별 브로커
  • OnRealtimeEtfNavReceived (Type: 0G) - 실시간 ETF NAV
  • OnRealtimeExpectedTradeReceived (Type: 0H) - 실시간 예상 체결
  • OnRealtimeInternationalGoldPriceReceived (Type: 0I) - 실시간 국제 금 가격
  • OnRealtimeIndustryIndexReceived (Type: 0J) - 실시간 업종 지수
  • OnRealtimeIndustryChangeReceived (Type: 0U) - 실시간 업종 변동
  • OnRealtimeStockInfoReceived (Type: 0g) - 실시간 주식 정보
Derivatives & System Events
  • OnRealtimeElwTheoreticalPriceReceived (Type: 0m) - 실시간 ELW 이론 가격
  • OnRealtimeMarketOpenTimeReceived (Type: 0s) - 실시간 장 개설 시간
  • OnRealtimeElwIndicatorReceived (Type: 0u) - 실시간 ELW 지표
  • OnRealtimeProgramTransactionReceived (Type: 0w) - 실시간 프로그램 매매
  • OnRealtimeViEventReceived (Type: 1h) - 실시간 VI 이벤트
Condition Search Events
  • OnConditionSearchListReceived - 조건검색 리스트 수신
  • OnConditionSearchRequestReceived - 조건검색 요청 수신
  • OnConditionSearchRequestRealtimeReceived - 실시간 조건검색 요청 수신
  • OnConditionSearchRequestRealtime2Received - 실시간 조건검색 요청2 수신
