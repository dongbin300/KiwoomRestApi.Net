API 개요

KiwoomRestApi.Net은 키움증권 OpenAPI REST API를 .NET 환경에서 쉽게 사용할 수 있도록 래핑한 라이브러리입니다.

주요 클라이언트 클래스

  • KiwoomRestApiClient: 메인 클라이언트 클래스
  • KiwoomSocketClient: 실시간 WebSocket 클라이언트

네임스페이스 구조

  • KiwoomRestApi.Net.Clients: 클라이언트 클래스들
  • KiwoomRestApi.Net.Enums: 열거형 정의
  • KiwoomRestApi.Net.Objects.Models: 데이터 모델
  • KiwoomRestApi.Net.Configuration: 설정 관련

OAuth 인증

GetOAuthTokenAsync ASYNC
OAuth 액세스 토큰을 발급받습니다.
public async Task<KiwoomRestApiResponse<KiwoomOAuthGetAccessToken>> GetOAuthTokenAsync( CancellationToken cancellationToken = default )

매개변수

매개변수타입설명기본값
cancellationToken CancellationToken 요청 취소 토큰 default
RevokeAccessTokenAsync ASYNC
OAuth 액세스 토큰을 무효화합니다.
public async Task<KiwoomRestApiResponse<KiwoomOAuthRevokeAccessToken>> RevokeAccessTokenAsync( string token, CancellationToken cancellationToken = default )

매개변수

매개변수타입설명기본값
token string 무효화할 액세스 토큰 -
cancellationToken CancellationToken 요청 취소 토큰 default

데이터 모델

주요 응답 모델

KiwoomRestApiResponse<T>

모든 API 호출의 기본 응답 구조

  • ReturnCode - 응답 코드
  • Message - 응답 메시지
  • Output - 응답 데이터
  • Output1 - 추가 응답 데이터
  • Output2 - 목록 데이터

KiwoomStockInfoGetStockCurrentPrice

주식 현재가 정보

  • StockCode - 종목코드

  • StockPrice - 현재가
  • PriceDifference - 전일대비
  • ChangeRate - 등락률
  • Volume - 거래량

KiwoomChartGetDailyCharts

  • BidQuantity1 ~ BidQuantity10 - 매수잔량1~10
