KiwoomRestApi.Net

The modern, intuitive, and fully asynchronous .NET wrapper for the Kiwoom REST API.

Get Started View API Reference

Why KiwoomRestApi.Net?

Fully Asynchronous

Leverage modern C# async/await patterns for non-blocking, high-performance API calls.

Comprehensive Coverage

Extensive support for a wide range of APIs, including account info, charting, orders, and real-time data via WebSockets.

Intuitive & Type-Safe

Designed with a fluent API and strong typing to minimize errors and improve developer experience.

Getting Started

1. Installation

Install the library via NuGet Package Manager.

PM> Install-Package KiwoomRestApi.Net
2. Basic Usage

Initialize the client and start making API calls.


using KiwoomRestApi.Net.Clients;
using KiwoomRestApi.Net.Enums.StockInfo;

// Initialize the client (replace with your keys)
var client = await KiwoomRestApiClient.CreateAsync("YOUR_APP_KEY", "YOUR_SECRET_KEY");

// Fetch stock information for Samsung Electronics
var stockInfo = await client.StockInfo.GetStockInfoAsync(
    stockCode: "005930",
    date: DateTime.Today,
    marginLoanType: KiwoomStockInfoMarginLoanType.Loan
);

if (stockInfo.IsSuccess)
{
    Console.WriteLine($"Stock: {stockInfo.Data?.StockName}");
    Console.WriteLine($"Current Price: {stockInfo.Data?.CurrentPrice:N0} KRW");
}
else
{
    Console.WriteLine($"Error: {stockInfo.ErrorMessage}");
}
